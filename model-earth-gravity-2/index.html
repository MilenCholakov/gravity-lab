Earth gravity forces and parts movement simulation

[ ФИЗИЧЕСКО ЯДРО: GRAVITY CORE V1.06 ]

Моделът симулира поведението на 2,000 дискретни маси, всяка от които представлява сегмент от планетарното тяло. За разлика от стандартните частици, тук всяка единица притежава собствена гравитационна сигнатура, влияеща на останалите 1,999 обекта в реално време.

Математическа рамка на взаимодействието:

$$\mathbf{F}_{ij} = G \frac{m_i m_j}{|\mathbf{r}_{ij}|^2 + \epsilon^2} \hat{\mathbf{r}}_{ij}$$

За поддържане на стабилността на „планетарната кора“ се използва Softening Factor ($\epsilon^2$). Той предотвратява безкрайното нарастване на силата при критично сближаване, симулиракйки структурната устойчивост на материята и предотвратявайки математически сингулярности.

Анализ на ускорението (FEM Spectrum)

 
Ниска активност (Blue): Векторна сума на силите $\approx 0$. Обектите са в състояние на хидростатично равновесие.
 
Стабилен преход (Green): Балансирано движение. Зона на формиране на стабилни пластове.
 
Висока динамика (Yellow): Зони на интензивно гравитационно привличане и ускорение на масите.
 
Критичен интензитет (Red): Максимално натоварване. Наблюдава се в центъра на акреция и при директни сблъсъци на масите.

Софтуерна Архитектура и GPU Оптимизация

Изчисляването на гравитационните взаимодействия между всяка двойка обекти има сложност $O(N^2)$. При 2,000 клетки това означава над 4 милиона калкулации на всяка под-стъпка. За постигане на обработка в реално време, изчисленията са изнесени към графичния ускорител (OpenCL/CUDA) посредством високооптимизиран паралелен конвейер.

Sub-Stepping (8x) Всеки графичен кадър съдържа 8 вътрешни физически итерации. Това гарантира, че бързодвижещите се частици няма да „прескочат“ през съседните си обекти, запазвайки коравината на планетарното тяло.
Contact Physics Използва се алгоритъм за геометрично отместване (Overlap Correction). При допир обектите се раздалечават векторно, симулирайки твърда повърхност без излишно софтуерно забавяне.

[ Изчислително C++ Ядро / GPU Симулация ]

void ComputeGravityStep() {
    for (int i = 0; i < totalCells; i++) {
        Vector2D totalForce = {0.0, 0.0};
        
        // Масивно паралелно пресмятане на взаимодействията (All-to-All)
        for (int j = 0; j < totalCells; j++) {
            if (i == j) continue;
            
            double deltaX = earthCells[j].posX - earthCells[i].posX;
            double deltaY = earthCells[j].posY - earthCells[i].posY;
            double distSq = deltaX * deltaX + deltaY * deltaY + softeningSq;
            double distance = sqrt(distSq);
            
            double forceG = (G_CONST * earthCells[i].mass * earthCells[j].mass) / distSq;
            totalForce.x += forceG * (deltaX / distance);
            totalForce.y += forceG * (deltaY / distance);
        }
        
        // Компенсация за центробежно ускорение и вискозно триене
        totalForce += CentrifugalForce(earthCells[i], omega);
        ApplyPhysics(earthCells[i], totalForce, damping);
    }
}
